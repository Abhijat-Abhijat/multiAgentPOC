🤖 QuantaBotics — Multi-Agent Intelligence Hub

QuantaBotics is a modular FastAPI-based AI assistant system that routes user inputs to intelligent agents for summarizing, calculating, coding, searching, visualizing, and more.

🚀 Getting Started

Install dependencies:
pip install fastapi uvicorn jinja2

📁 Project Structure

project/

⚙️ API Endpoints

GET /

Serves the homepage (HTML UI).

POST /run

Processes user input via agent dispatch.

{
  "input": "code: write a function to reverse a string"
}

Sample Response

{
  "log": [...],
  "final": "def reverse_string(s): return s[::-1]"
}

🧠 Input Routing Logic

User inputs are routed by prefix to the correct agent:

PrefixRouted ToAgent Name
code:Code Generatoragent5_code_writer
search:Web Searcheragent6_web_searcher
sentiment:Sentiment Analyzeragent7_sentiment_analyzer
visualize:Data Visualizeragent8_data_visualizer
events:Event Extractoragent9_event_extractor
keywords:Keyword Extractoragent10_keyword_extractor
Math expressionCalculatoragent4_calculator
DefaultSummarizer → Formattersummarizer → formatter_agent

📨 Agent Message Format

Each agent receives messages in this format:

{
  "sender": "agent1",
  "receiver": "agent5_code_writer",
  "task": "generate_code",
  "content": "code: reverse string"
}

🛠️ Adding a New Agent

  1. Create agents/agent11_language_translator.py
  2. Define a function agent11_task(msg)
  3. Add it to agent_registry in main.py
  4. Add input condition to agent1_task()
